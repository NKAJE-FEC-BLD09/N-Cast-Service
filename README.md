# N-Cast-Service

Technical Requirements/dependencies
 - Client: React, Webpack/Babel, Bootstrap 
 - Server: Express, Nodemon 
 - Database: PostgreSQL 

To Get Data:
install postgres and then run the script 'npm run database'

To Run Application:
 - run server script
 - run build script
 - exists on localhost:3003

Features Include:
